# Change Log

All notable changes to the "codemate" extension will be documented in this file.

Check [Keep a Changelog](http://keepachangelog.com/) for recommendations on how to structure this file.

## [0.1.0] - 2025-12-20
### Added
- Initial release.
- Base64 helper tool for encoding and decoding strings.

## [0.2.0] - 2025-12-27
### Added
- Introduced json to code generation for TypeScript, C#, Python, Java, Go, JavaScript, Kotlin, and Ruby.

## [0.3.0] - 2025-12-31
### Added
- Introduced code to json generation for TypeScript, C#, Python, Java, Go, JavaScript, Kotlin, and Ruby.
